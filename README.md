Pokémon Battle Arena
Pokémon Battle Arena es una simulación de batallas Pokémon desarrollada por un fan, que busca recrear la experiencia de combate de la franquicia original.
Características
- Simulación de batallas: Reproduce las mecánicas de combate de los juegos de Pokémon, permitiendo a los jugadores enfrentarse en duelos estratégicos.
- Interfaz amigable: Diseñada para ser intuitiva y accesible, facilitando la inmersión en las batallas.
- Desarrollo en JavaScript: Utiliza tecnologías web estándar como JavaScript, HTML y P5 para su implementación.
Instalación
1. Clonar el repositorio:
   git clone https://github.com/PabloValdazoDEV/Pokemon-Battle-Arena.git

2. Navegar al directorio del proyecto:
   cd Pokemon-Battle-Arena

3. Abrir el archivo `index.html` en un navegador web para iniciar la aplicación.
Uso
- Selección de Pokémon: Elige tu Pokémon para la batalla.
- Ejecución de movimientos: Selecciona los movimientos durante el combate para derrotar al oponente.
- Estrategia: Utiliza tácticas efectivas para ganar las batallas.
